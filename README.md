# 한국 지자체별 지방선거 투표율 Dashboard

중앙선거관리위원회와 공공데이터포털 공식 원본을 바탕으로 시군구 단위 지방선거 투표율을 정리한 정적 대시보드입니다.

현재 포함 범위:

- 제3회 전국동시지방선거: 2002-06-13
- 제4회 전국동시지방선거: 2006-05-31
- 제5회 전국동시지방선거: 2010-06-02
- 제6회 전국동시지방선거: 2014-06-04
- 제7회 전국동시지방선거: 2018-06-13
- 제8회 전국동시지방선거: 2022-06-01

핵심 파일:

- `index.html`: GitHub Pages에 바로 올릴 수 있는 대시보드
- `data/local-election-turnout-municipal.csv`: 다운로드용 정규화 데이터
- `data/local-election-turnout-municipal.json`: 대시보드 구동 데이터
- `data/sources.json`: 원천 링크 목록
- `scripts/build_data.py`: 공식 원본 다운로드와 데이터 생성 스크립트

재생성 방법:

```bash
python scripts/build_data.py
```

원천:

- 중앙선거관리위원회 자료공간 `제3회~제6회` 개표결과 묶음
- 공공데이터포털 `제5회~제8회` 전국동시지방선거 개표결과 원문파일

메모:

- 대시보드는 시군구 단위 전체 투표율 비교를 위해 각 회차의 `시·도지사` 기준 집계값을 사용합니다.
- 제3회 자료는 중앙선관위 레거시 압축 묶음 안의 광역 `xls` 파일에서 시군구 `합계` 행을 추출해 정규화했습니다.
- 2006년 제4회 자료는 중앙선관위 레거시 `xls` 원본을 정규화해 포함했습니다.
- 2002년 제3회는 총 243개 시군구가 잡히며, 이후 신설·분구된 지역 때문에 2006년 이후 회차와 개수가 다릅니다.
